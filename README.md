# vim-github-dark

A dark GitHub theme for [Vim](https://www.vim.org/) and [NeoVim](https://neovim.io/).

## Install
To install using [vim-plug](https://github.com/junegunn/vim-plug) add this line to your `vimrc`:  
```Plug 'wojciechkepka/vim-github-dark'```
